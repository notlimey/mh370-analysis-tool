# Knowledge Base

Primary source documents for the MH370 analysis tool. Scripts fetch, parse, and extract data from official reports and papers so that any contributor can rebuild the knowledge base from scratch.

## Directory structure

- `papers/raw/` — Original PDFs downloaded by `fetch_sources.py`. Git-ignored; run the fetch script to populate.
- `papers/parsed/` — Markdown versions of PDFs produced by `parse_sources.py`. Git-ignored; run the parse script to populate.
- `extracted/` — Specific verified data points extracted from parsed papers. Git-ignored; run extraction scripts to populate.
- `sources.md` — Manifest of all source documents with URLs, checksums, and download status. Generated by `fetch_sources.py`.

## Rebuilding from scratch

```bash
# 1. Install Python dependencies
pip install pymupdf pdfplumber requests

# 2. Download all source PDFs
python knowledge/fetch_sources.py

# 3. Parse PDFs to searchable markdown
python knowledge/parse_sources.py

# 4. Extract specific data points
python knowledge/extract_dstg_18_28.py
```

## What is committed

Only the scripts and documentation are committed:
- `README.md` — This file
- `sources.md` — Download manifest (regenerated by fetch script)
- `fetch_sources.py` — Downloads PDFs
- `parse_sources.py` — Converts PDFs to markdown
- `extract_dstg_18_28.py` — Extracts 18:28 position data from DSTG book
