# Sonar Coverage GeoJSON Files

Coverage polygons extracted from the AusSeabed WMS server, showing the areas scanned during the MH370 underwater search.

## Files

| File | Description | Features |
|------|-------------|----------|
| `auv_coverage.geojson` | AUV (Autonomous Underwater Vehicle) side scan sonar, 5m resolution | 43 polygons |
| `deeptow_coverage.geojson` | Deep tow side scan sonar, 5m resolution | 2 polygons |
| `gophoenix_coverage.geojson` | Go Phoenix / Dong Hai Jiu 101 synthetic aperture sonar, 5m resolution | 12 polygons |
| `bathymetry_coverage.geojson` | Bathymetry survey, 150m resolution (wider area) | 36 polygons |
| `all_coverage.geojson` | Union of all four layers into a single MultiPolygon | 1 feature |

|----|-----------|

| deeptow | `ausseabed:MH370_Phase_2_Sonar_Imagery_Backscatter_Inverse_Deep_Tow__SSS__5m_2018` |

**Note:** No dedicated "Go Phoenix" layer exists on the AusSeabed WMS. The DHJ (Dong Hai Jiu) SAS wide backscatter layer covers data from the Go Phoenix / Dong Hai Jiu 101 survey vessel.

## Extraction Method

Generated by `scripts/extract_sonar_coverage.py` on 2026-04-08.

### Parameters

- **Bounding box:** 85E to 100E, 42S to 20S
- **Image size:** 1500 x 2200 pixels
- **WMS version:** 1.1.1 with EPSG:4326, lon/lat BBOX ordering
- **Mask threshold:** alpha > 0 (any non-transparent pixel)
- **Morphological cleanup:** 3-pixel dilation + erosion (binary close) to merge adjacent scan swaths
- **Minimum contour area:** 50 pixels
- **Polygon simplification:** 0.02 degrees (~2 km) for sonar layers, 0.03 degrees for bathymetry
- **Format:** GeoJSON (EPSG:4326)

### Process

1. Request a single GetMap PNG with transparency from the WMS for the full bounding box
2. Extract alpha channel to create a binary mask (data vs. no-data)
3. Apply morphological close (dilate then erode) to fill small gaps between scan swaths
4. Extract contours at the 0.5 level using scikit-image
5. Filter out contours with area < 50 pixels
6. Convert pixel coordinates to lon/lat using the linear mapping from the bounding box
7. Simplify polygons with Shapely to reduce file size
8. Save individual and combined GeoJSON files

## How to Regenerate

```bash
# Install dependencies
pip install requests Pillow numpy scipy shapely scikit-image geojson

# Run from project root
python3 scripts/extract_sonar_coverage.py
```

Output will be written to `public/data/sonar_coverage/`.
